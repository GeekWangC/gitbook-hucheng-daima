# 第二十四章 小齐离开

那天下午的光很薄，像贴在窗上的一层旧胶片。小齐把工位上的绿植分给大家，一盆给前端组，一盆给测试，一盆给后台。他把最后一盆放到窗边，手指在叶片上停了一下。

“我去另一家公司了。”他说，“离这儿不远。”

没有人惊讶。过去一周，他在工位上写了厚厚一叠“交接笔记”。不是流程截图，不是堆叠的链接，而是一张张能被直接使用的“行动单”：

- 如果 SSR 渲染在预发偶发失败，请先看 ADR #42 的‘薄雾’说明；
- 如果端到端测试 flaky，请按‘三句事实’贴到白板；
- 如果用户在某一步骤停顿，请先检查‘词—句—触点’清单。

“还有这个。”他把笔记本最后一页撕下，递给陈砺。上面写着：

“把快放在秩序里。”

送别那天，他们没有去 KTV，也没有去吃火锅。他们在楼下的小公园里坐了一会儿。风从黄杨树间掠过，带着一点土的香味。小齐看着他们，说了最后一段话：

“我来这儿时，总想快；我要走了，知道了怎么把快变成一种习惯。我写了十几个 PR 模板，改了四个 pipeline，把一堆‘偶发’变成了被命名的‘薄雾’。这些东西有没有用，我不知道；但我知道，我能把它们带走，也能把它们留在你们这儿。”

“远程回流文化，不要丢。”他看着陈砺，“不是每个建议都要被合并，但每一次远程的对话，最好都有回流。”

所谓回流，是指每一份外部贡献（哪怕只是一个意见），都应该获得看得见的回应：被采纳、被搁置、被反对，都以事实描述，并给出下一次复议的时间。

“这是防止‘热情耗损’最简单的办法。”江牧说，“把每一份尝试当成一张账单——不是要结账，而是要记账。”

小齐离开后，团队把“回流文化”落实为几个小动作：

1. PR 模板里新增“回流说明”一栏；
2. 每周例会最后五分钟，回顾未处理的外部反馈；
3. 在知识墙上加了一列“回流完成”卡片，完成一次，翻一张。

周五晚上，小齐发来第一封“远程 PR”。改的是文案，把一个“抱歉”换成了“我们已经收到”。

“这句更像行动。”他在 PR 里写。

第二天，PR 被合并，卡片被翻成了背面。墙边经过的人多看了一眼。

“送你到地铁站？”陈砺问。

“不用。”小齐笑，“我自己去。”

他背着包走向地铁口，肩膀有点瘦。黄昏把他的影子拉得很长。进站之前，他回头向他们挥了挥手，像每一次上线之前，在群里发的那句“准备好了”。

回到办公室，陈砺把小齐留下的“行动单”装进透明文件夹，放在知识墙的第二层。夹子上贴了一张小标签：

“把快放在秩序里。”

之后很长一段时间，团队仍时常在群里看到“小齐”的头像。他会在深夜发来一张截图，或者在清晨留下一条评论。没有人问他的近况。他们只是像以往一样，用事实回应，用行动回流。

城市的夜一点一点落下来。有人在巷口买花，有人在小店门口喝酒，有人在窗前拆信。每个人都在离开，每个人也都在抵达。

“写信吧。”苏棠说，“写一封‘合唱信’给他。”

于是他们一起写：第一段写他们做过的事情，第二段写彼此学到的东西，第三段写未来的节奏。信不长，句子短，像他们最近写的每一条日志。落款是一群人的名字，像一首没有歌词的合唱。

信寄出的时候，风正好吹过来。纸张在风里微微起伏，像这座城市里无数个看不见的细小变更。它们不吵不闹，却会在某个清晨，让你发现：秩序，已经悄悄改变了你的呼吸。

几周后，远程回流文化开始显露它的力量。一个周末的清晨，小齐提了一个很小的 PR：把某个错误提示里的“抱歉”换成了“我们已经收到”；又把“请稍后再试”换成了“你可以稍后再试，或在这里查看状态页”。这两个小改动在一周内让客服的“重复询问”减少了 8%。

“语言也是工程。”陈砺在周会上说。

他们把“回流”做成流程的一部分：

- 每个 PR 都有一个“回流摘要”栏，写清楚这次改动的来源：内部发现/外部建议/用户反馈；
- 每周五的“回流时间”固定 30 分钟，回看未回复的外部讨论；
- 对每一条无法采纳的建议，必须给出证据与“下次复议”的时间。

一个月后，知识墙上“回流完成”的那一列，像呼吸一样有节奏地翻动着卡片。没有人再觉得“回复一条评论”是负担，它更像是一种礼貌，一种对彼此时间的尊重。

有一次，一个远程贡献者在凌晨两点提交了一个修复，修复的是一个很少被人注意到的边界条件。一觉醒来，团队就在晨会上看到了它。那天所有人的“晨会一句话”都很短：

“谢谢。”

午后，阳光从百叶窗的缝里斜斜地照进来。陈砺看着那些卡片，忽然意识到：小齐离开带走了一个人，却留下了一种风。风不在嘴上，它在墙上，在卡片里，在每一条“回流完成”的翻面声里。

傍晚，小齐给他发了一条语音。

“最近好吗？”

“挺好。”

“我那边也还行。我们刚把‘对齐日’搬了过去，大家第一次参加还有点不习惯，但写‘新闻稿’的时候，都认真起来了。”

“认真是会传染的。”陈砺说。

挂断电话，他在墙上贴了一张新的卡片：

“远程回流的黄金法则：任何一次善意的尝试，都值得一个清晰的回应。”

夜里，他经过地铁站，玻璃门上映着一列列进站的灯。每个人背着各自的包，朝着各自的方向走去。有人会离开，有人会回来；有人会在远处接续一条未完的线。回流不是把人拉回来，而是让线一直在。

临睡前，他在笔记里补了一句：

“回流让‘离开’变成‘继续’。”

第二天清晨，办公室的第一缕阳光照到知识墙上。墙上新贴的那张卡片在光里发亮，像是在说：欢迎每一阵来自远方的风。

午间吃饭的时候，大家翻起旧相册，看到小齐刚来时坐在角落里、手里捏着一杯纸咖啡的样子。有人笑出声来，有人红了眼眶。陈砺把相册合上，说：“我们今天继续。等他哪天回来，我们把新的卡片念给他听。”这种念给离开者听的仪式，不是伤感，而是把一条线拉得更直一点。

傍晚，窗外的光线慢慢倾斜。知识墙前又站了两个人，他们在卡片上各自写下了今天的回流。笔尖在纸上走过的声音细细碎碎，像城市在远处缓慢运转的回响。

又过了一个月，小齐作为“远程教练”带了一位新人做了三次异步 Pair。异步 Pair 的规则很简单：

1. 用一张卡片写“我现在卡在这里”；
2. 用一句话说“我下一步会做这个”；
3. 用一个链接贴“我参考了这个证据”。

三次之后，那位新人在群里说：“我第一次感觉远程并不遥远。”

他们把这套规则收进了“回流手册”的末尾。手册不厚，十页不到，封面是一只小纸飞机。翻开第一页，写着：

“回流，是把善意送达。”

周末的午后，阳光落在木地板上，有点暖。他们在会议室里摆了两排椅子，办了一次小小的“回流茶会”。没有议程，每个人带一张卡片来，念一念这个月最想感谢的那一次回流。有人念了远方的修复，有人念了走廊的便签，有人念了那封众人合写的合唱信。念到最后，大家都笑了。笑声轻盈，却有重量。

“回流也是让我们不孤单的方法。”陈砺说，“它让‘离开’不再是消失，而是另一种出现。”
