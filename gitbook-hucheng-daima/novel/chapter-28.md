# 第二十八章 冬至

冬至这天，公司食堂破例多了一道汤圆。窗外的风收敛了锋利，像被包裹的边角。办公室里，谁的桌上都多了一点甜。

“今天不合 PR 了。”陈砺在群里发，“写写年终总结，给自己放一个小时的空。”

他们在会议室里做了一次轻量的年终回顾：每个人写下“今年最想保留的一件事”和“明年最想改变的一件事”。纸条贴在白板上，像一个个小小的锚点。

轮到他时，他写下：保留“把问题翻译成语言”的习惯；改变“在压力下抓得太紧”的反射。

傍晚，他坐在窗边，写下这一年的关键词：秩序、适配、承诺、看见、放下。慢，并不是倒退。它是系统主动选择的节奏，是为了更稳地走更远。

夜色里，愚园路的书店灯仍然亮着。苏棠发来一张照片：两碗汤圆，蒸汽在镜头前模糊成一层柔光。他忽然很想回去，坐在那盏台灯下，什么也不做。

他照做了。回到店里，苏棠把一条薄毯搭在他腿上。他们不说话，只听外面的风偶尔推门一指。关门前，他在小黑板上写下四个字：冬至快乐。
