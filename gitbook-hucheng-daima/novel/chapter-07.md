# 第七章 风暴中的回归测试

风暴早至：预算收紧，名额冻结，项目合并。有人离开，有人沉默，笑语间多了自嘲。

“把人当模块，会更难受。”某个黄昏，陈砺独对空座，心下自语。

他拉起一纸“回归清单”，不是给代码，是给人：

- 每周一场 Demo Day，再小也要有可见进度；
- PR 描述用“动机—方案—影响”三段式；
- 文档与自动化的预算不减；
- 一对一谈话，确认每个人的担忧。

第三周，办公室风扇声小了，桌边笑声多了。秩序未必挡得住风暴，却能教人知所系。

# 第七章 风暴中的回归测试

风暴早至：预算收紧，名额冻结，项目合并。有人离开，有人沉默。走廊里的灯比往常亮一点，像为了抵御什么而故意调高了亮度。午后茶水间的笑声稀薄，大家说话都挑着词，生怕踩到看不见的边。

“把人当模块，会更难受。”某个黄昏，陈砺独对空座，心下自语。电脑屏幕倒映着他自己的脸，他忽然发现，比起“代码的回归”，人更需要“节奏的回归”。

他拉起一纸“回归清单”，不是给代码，是给人：

- 每周一场 Demo Day，再小也要有可见进度；
- PR 描述用“动机—方案—影响”三段式；
- 文档与自动化的预算不减；
- 一对一谈话，确认每个人的担忧；
- 每人一张“红线卡”，当感到超负荷时可以亮出，无须解释。

第一周，清单像放在桌面的一张白纸，只有几个人试着照着做。第二周，Demo Day 上出现了一个几乎看不见的小变化：每个演示的最后都多了一页“下一步是什么”。第三周，林越把“红线卡”贴到了每个座位的隔板上，卡片中央印了一个很小的“停”字。她说：“看见它的时候，就允许自己停一停。”

团队里有一个同事，名叫韩舟，向来沉默，是那种“开了机就不说话”的人。这段时间，他的提交频率先抬后落，像一条不愿意被打扰的曲线。某天夜里，小齐发现他把一个关键模块独立重写，却没有开分支也没有文档。第二天早上，陈砺叫他到窗边：“说说看，是不是哪儿让你觉得‘来不及解释’？”韩舟沉默了很久，说：“我怕我解释得不够好。”陈砺说：“解释不是考试，记录不是审判。我们把‘说不清’也写下来。”

那天的晚些时候，他们开了一场“节奏课堂”。没有 PPT，只有几根彩色的记号笔和一面白板。陈砺把“节奏”拆成五个字：看、慢、分、写、停。看，是先看事实；慢，是容许自己慢一点；分，是把问题拆小；写，是写下来的约束与承诺；停，是当红线卡出现时，允许自己停一停。小齐问：“那‘快’去哪了？”陈砺说：“快在‘分’里，快在‘写’里，快在‘已经练过很多次’里。”

风暴并没有因此离开。第四周，合并后的第一个联动需求压下来，时间线紧得像把弦。第一天的站会，大家把要做的模块列满一面墙；第二天，依赖关系一交叉，就卡住了。小齐想要“先动手再说”，陈砺按住他的肩，带他把每一条依赖都画成箭头，最后在白板的右下角留下两行字：

1. 当依赖链长于三时，先写 ADR；
2. 当 ADR 不被理解时，先写“给非技术看的解释”。

第三天晚上，Ops 报了一条“细小的异常”：某个页面在凌晨的冷流量下错误率抬升，抬升幅度不大，却像心电图上突然竖起的一根细线。小齐和韩舟熬到午夜，终于定位到是一个边角的缓存过期策略与新引擎的懒加载策略叠在了一起。当他们把策略调回去、错误率落下时，窗外苏州河的风正徐徐吹过，像把这一夜的紧张轻轻抚平。

紧接着，他们做了一次“风暴中的回归测试”演练：

- 每个小组列出过去一个月里最害怕的三件事；
- 把每件事改写成“可以被观察”的三条指标；
- 为每个指标设计一个“慢变量”——在压力下仍然可以坚持的小动作。

林越写下的三条慢变量是：每日同步十分钟、每周一页“给业务看的简报”、每月一次“说话不用技术词”的分享。她说：“慢变量会给我一种稳稳的感觉。”

一位即将离开的同事递交了离职申请。最后的午餐，他坐在窗边，说：“我不是不喜欢这里。我只是害怕‘一直快下去’会看不见路。”陈砺说：“谢谢你告诉我。我们会把‘慢’写进方法里，而不是写在口头上。”

当晚，陈砺把“节奏课堂”的五个字、林越的三条慢变量、以及这位同事的那句“看不见路”的担心，写成了一篇短文，题为《看见彼此，才看见路》。文末是一句很轻的话：我们会在风里走很久，但不要忘了相互看一眼。

第五周，团队开始恢复某种“呼吸感”。不是每个人都在同一个节拍，但每个人都在自己的节拍里稳住了脚。Demo Day 上，有人展示了一个看似微不足道的改进：在一处常被忽略的对话框里加上了“键盘逃离”的能力。所有人都鼓了掌。那一刻，没有人谈“效率”，也没有人谈“KPI”。他们谈的是“好好对待每一个人”。

周末的午后，陈砺坐在愚园路的店里，笔记本上写着新一周的计划。一阵风吹过，把门口的风铃轻轻撞响。他抬起头，看到苏棠正把一束新花放进玻璃瓶里。花在水里慢慢张开，像一个系统在慢慢恢复自己的节奏。他合上本子，心里很安静。

他知道，风暴还会来。但他们已经学会了：当风起时，不是每个人都要站成同一种姿势；他们只需要记得“看、慢、分、写、停”，再把彼此扶一把，就能从容地走过下一段路。
