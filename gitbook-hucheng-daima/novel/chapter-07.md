# 第七章 风暴中的回归测试

风暴早至：预算收紧，名额冻结，项目合并。有人离开，有人沉默，笑语间多了自嘲。

“把人当模块，会更难受。”某个黄昏，陈砺独对空座，心下自语。

他拉起一纸“回归清单”，不是给代码，是给人：

- 每周一场 Demo Day，再小也要有可见进度；
- PR 描述用“动机—方案—影响”三段式；
- 文档与自动化的预算不减；
- 一对一谈话，确认每个人的担忧。

第三周，办公室风扇声小了，桌边笑声多了。秩序未必挡得住风暴，却能教人知所系。
