# 第一章 热更新的清晨

清晨六点五十五分，张江的风从小区的人行栅栏缝里挤进来，像浏览器里一条未压缩的长请求，冷冷地贴在皮肤上。陈砺把电脑合上，双肩包背好，掐灭最后一盏台灯。屏幕上停着昨天的代码评审：一个 PR 标题写着“feat: migrate to Vite”。

三十五岁，前端第十二个年头，他很熟悉这种迁移带来的阵痛：包管理器，构建链路，边界条件，被遗忘的 polyfill，和两三个已经离职的作者。陈砺把门轻轻一带，电梯门在走廊尽头亮起白光，像一个刚刚启动的 dev server。

地铁二号线，往广兰路方向。窗外的高架桥一截一截后退，像时间被拆成了可复用的组件。车厢里，穿衬衫的人低头刷邮件，穿卫衣的人盯着手机屏保，所有人都匀速滚动，像一页页无限下拉的 feed。到世纪大道换乘，风从站厅吹下来，明晃晃地带着咖啡豆和烘烤面包的味道。

公司在张江一栋灰白的楼里，玻璃幕墙对着一条短短的人工河。保安识得他，点点头。前台花瓶里插着白色的桔梗，刚换的水还带着微微的消毒水味。九点一刻，技术周会准时开始，屏幕投出一张又一张路线图：增长，闭环，ToB，AI Copilot，合规。每个关键词像一枚小小的锚点，挂在他的胸腔里，随呼吸起伏。

“砺哥，Vite 切完，老活动页还能跑不？”新来的校招生小齐在会后追上他，眼睛里带着不确定。

“我们做个适配层，不要把历史丢了。”陈砺笑笑，指着白板，“别急着删，先把依赖关系梳理清楚。热更新是给代码的，稳定是给人的。”

热更新。这个词从他嘴里滑出去，像一枚温热的石子，落在水面，一圈一圈荡开。三十五岁，身边的朋友开始换赛道：有的转管理，有的去做独立开发，有的回老家开了一家咖啡馆。比起“升降级”，他更愿意把这些叫作“迭代”。

午后，阳光从百叶窗缝里斜斜地打进来，像一行行注释。陈砺打开一个旧项目，里面是五年前他写的代码，eslint 规则还是当年的样子，变量名像急促的脚步，注释像深夜的喘息。他盯着一段奇怪的 Hack，忽然觉得它像一条被封存的短信：你当时也很用力地过生活啊。

“下午两点，二号会议室。”HR 发来一条消息，语气平平。走进会议室时，窗外正好飘起一阵极轻的雨，像 CSS 的淡入动画。

“组织要调整，前端会合并到平台组，”HR 说，“你的 title 不变，但要兼顾社区组件库和内部工程化。”

“我知道。”陈砺点点头。消息在他心里落地，像一个提前知道的 breaking change。不是坏事，只是要重写几个入口文件。

会后路过苏州河。雨停了，水面灰白，偶尔有一片树叶被涌动推着往前。远处的新楼像从模板里倒出来，统一而干净。风过桥洞时带起一点潮气，像旧城里不肯散去的 commit 历史。

傍晚，他从陆家嘴地铁站出来，想走一走再回去。玻璃写字楼把天空切成多块，灯逐一亮起，仿佛一个个路由被激活。街角店铺的门敞开着，油烟里裹着葱花的香气，像小程序里的轻弹窗，不打扰但也不退场。陈砺想起多年前刚来上海的自己，住在愚园路的一间老公房，夏天没有空调，窗外有梧桐也有蝉鸣。夜里趴在桌上学 React，写简历时把“熟悉”改成了“精通”，心里有数却也不怕。

他给小齐发了一条消息：“今晚我把兼容方案写个 RFC，你先拉个 demo 仓库，不要用公司模板，用最轻的。”

消息发出，手机屏幕黯下来，倒映出他的脸。三十五岁并不老，只是需要承认速度没那么快了。热更新不是无所不能，它也需要边界：当状态复杂到一定程度，刷新一次也许更好。可人生不能重启，能做的只是把变化拆小，一次处理一个 diff。

夜里回到家，他把电脑放到餐桌上，窗外有远处高架的车流声，像后台服务器稳定的心跳。陈砺打开一个空白的文档，在标题栏敲下几个字：沪城代码 · 计划。下面列出三条简单的 bullet：

1. 迁移构建链路，保留历史可运行。
2. 开放组件库，统一风格与文档。
3. 找回写代码的快乐。

他想了想，又在底下加了一行：和人好好说话。

屏幕的白光静静铺开，像一张干净的画布。他突然意识到，所谓“工程化”，从不是为了快，而是为了在变化里维持一种可以预期的秩序。这种秩序不是枷锁，而是让人能继续往前走的路径。窗外，风吹过梧桐，像某种看不见的事件流。他抬起手，开始打字。

那一刻，他听见了城市底层运行的声音：地铁，光，雨，人的脚步，代码热更新时短暂的停顿与跳跃。上海是一个巨大的运行时，而他仍是其中一个持续监听变化的进程。
